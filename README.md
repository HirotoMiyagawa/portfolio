# Portfolio

制作したWebアプリ・ツールをまとめたポートフォリオです。

シンプルに使えること、実際の生活や学習の中で役立つこと、GitHub Pagesなどで公開しやすい静的構成にすることを意識して制作しています。

## Projects

### Habit Tap

毎日の習慣を記録し、継続をサポートする習慣チェックアプリです。

達成、スキップ、メモ、週間カレンダー、リマインド通知などを備え、無理なく習慣を続けられるようにしました。

- 公開URL: https://hirotomiyagawa.github.io/Habit-Tap/
- GitHub: https://github.com/HirotoMiyagawa/Habit-Tap
- 詳細: [habit-tap-portfolio.md](./habit-tap-portfolio.md)

主な技術:

- HTML
- CSS
- JavaScript
- localStorage
- Browser Notification API
- GitHub Pages

### かんたん家計メモ

「今日いくら使ったか」をすぐ記録できるシンプルな家計メモアプリです。

日付、金額、カテゴリー、メモを記録でき、カレンダーで日ごとの支出や月ごとの合計を確認できます。カテゴリーは追加、編集、削除に対応しています。

- 公開URL: https://hirotomiyagawa.github.io/kakeibo-memo/
- GitHub: https://github.com/HirotoMiyagawa/kakeibo-memo
- 詳細: [kakeibo-memo-portfolio.md](./kakeibo-memo-portfolio.md)

主な技術:

- HTML
- CSS
- JavaScript
- localStorage
- GitHub Pages

### Pokemon Selection App

ポケモンの選出を支援するためのアプリです。

相手の構成やタイプ相性をもとに、自分の候補から有利な組み合わせを考える補助ツールとして作成しました。

- 詳細: [portfolio_pokemon_selection.md](./portfolio_pokemon_selection.md)

主な技術:

- Python
- Tkinter
- タイプ相性ロジック

## Skills

- HTML/CSS/JavaScriptによる静的Webアプリ制作
- localStorageを使ったブラウザ内データ保存
- カレンダー、一覧、フォームなどのUI実装
- GitHub Pagesでの公開
- ユーザーが継続して使いやすい機能設計
- 小さく作って改善するアジャイル的な開発

## Development Style

各アプリは、まずMVPを作成し、その後ユーザーの要望に合わせて段階的に改善しています。

例:

- 最初に基本機能を実装
- 実際にブラウザで動作確認
- 必要な機能を追加
- READMEやポートフォリオ資料を更新
- GitHub Pagesで公開
