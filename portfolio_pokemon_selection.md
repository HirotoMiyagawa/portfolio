# ポケモン選出支援ツール

## 概要

ポケモン対戦における選出判断を支援するWebアプリです。

自分のパーティと相手のパーティを入力すると、タイプ相性や参考技タイプをもとに、選出すべき3体と相手ポケモンごとの対面担当を判定します。

## URL

- Webアプリ: https://hirotomiyagawa.github.io/Pokemon-Team-Selection/
- GitHub: https://github.com/HirotoMiyagawa/Pokemon-Team-Selection

## 使用技術

- HTML
- CSS
- JavaScript
- GitHub Pages
- Google Search Console

## 主な機能

- 自分のパーティ6体を入力
- 相手のパーティ6体を入力
- 相手ポケモンは名前なし、タイプのみでも判定可能
- ポケモン名入力によるタイプ自動反映
- タイプ入力によるポケモン候補表示
- 参考技タイプをもとにした技範囲の考慮
- 攻撃評価、被ダメージ評価、技範囲補正による総合スコア算出
- スコア上位3体を自動選出
- 相手ポケモンごとの対面推奨表示
- 自身のパーティ登録、読込、削除
- ポケモン徹底攻略への育成論参考リンク表示
- 採用率データをもとにした対戦時の注意点表示
- SEO向けのmetaタグ、OGP、robots.txt、sitemap.xml対応

## 工夫した点

当初はタイプ相性のみで選出を判定していましたが、実戦では技範囲や採用率も重要になるため、参考技タイプや採用率に基づく注意点を追加しました。

また、入力の手間を減らすため、ポケモン名を入力するとタイプと参考技タイプが自動反映されるようにしました。タイプから候補を表示することで、名前が分からない場合でも入力しやすくしています。

データ保存はサーバーを使わず、ブラウザのlocalStorageを利用しています。これにより、無料のGitHub Pages上で公開しながら、パーティ登録機能を実現しました。

## 開発の流れ

1. Windows向けのローカルデスクトップアプリとして試作
2. Python/TkinterでGUI版を作成
3. Web公開のためHTML/CSS/JavaScript版に移行
4. GitHub Pagesで無料公開
5. Google Search Consoleに登録
6. SEO向けにmeta情報、OGP、sitemap.xml、robots.txtを追加
7. 技範囲や採用率に基づく注意点を追加
